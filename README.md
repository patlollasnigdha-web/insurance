# Insurance Management System (Java)

This is a beginner-friendly console-based Insurance Management System developed in Java.

## 📌 Features

- Add Insurance Holder
- Display All Insurance Holders
- Calculate Premium
- Check Policy Maturity
- File Handling (Data stored in .txt file)
- Exception Handling
- Array of Objects Implementation
- Constructor Usage

## 🛠 Technologies Used

- Java
- OOP Concepts
- File Handling (FileReader, FileWriter, BufferedReader)
- Exception Handling
- Arrays

## 💡 Concepts Covered

- Classes and Objects
- Constructors
- Methods
- Array of Objects
- File Handling
- Try-Catch Blocks
- Switch Case
- Basic Console UI
